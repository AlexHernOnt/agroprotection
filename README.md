# 🌾 AgroProtection

Aplicación web desarrollada en Java con JSF, JDBC y MySQL para la gestión de usuarios y terrenos agrícolas.

Arquitectura hexagonal (UI, Application, Domain e Infrastructure) con separación clara de responsabilidades.

Sistema de login con gestión de sesión mediante beans de JSF.

Visualización de terrenos asociados a cada usuario obtenidos desde base de datos.

Creación de nuevos terrenos persistidos en MySQL mediante PreparedStatements.
